### 2. 资产净值图表

**优化前**：
```javascript
// 基础配置
line: {color: '#667eea'}  // 固定紫色
margin: {t: 20, r: 40, b: 40, l: 60}
height: 400
```

**优化后**：
```javascript
// 专业配置
line: {
    color: '#3b82f6',      // 专业蓝
    width: 3,               // 更粗

fillcolor: 'rgba(59, 130, 246, 0.1)'
hovermode: 'x unified'      // 统一悬停
height: 450                 // 更高
```

**效果对比**：
