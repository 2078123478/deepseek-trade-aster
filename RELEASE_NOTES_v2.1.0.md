## 🎯 使用建议

### 日常监控
1. **每天查看性能指标** - 评估策略表现
2. **切换时间范围** - 观察长期趋势
3. **关注回撤分析** - 及时控制风险
4. **查看交易详情** - 分析AI决策

### 策略优化
- **胜率 < 50%** → 调整信号逻辑
- **盈亏比 < 1.0** → 优化止盈止损
- **回撤 > 15%** → 减小仓位
- **收益率为负** → 全面检查策略

## ⚠️ 已知问题

### Windows UTF-8编码
**问题**: 控制台可能显示乱码  
**解决**: 已在dashboard_app.py中修复
```python
sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf-8')
```

### 数据积累建议
**建议**: 为获得最佳展示效果
- 最少: 10笔交易记录
- 推荐: 50笔以上
- 最佳: 100笔以上
